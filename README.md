Bistutu空教室查询
===================

安卓版本。

## 项目简介

本软件适用于北京信息科技大学空教室查询，开发此软件的目的是为了能够帮助热爱学习的小伙伴迅速找到合适的空教室，软件设计理念追求简洁，软件目前正在持续维护中。

## 使用到的第三方库

```
1. QMUI 界面设计
2. okhttp3 网络请求
3. Gson JSON数据解析
```
